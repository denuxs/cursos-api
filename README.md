# Cursos API

API REST para registro de estudiantes y cursos

### Paquetes

* Python 3+
* Django 5
* Django Rest Framework
* Simple JWT
* Sqlite

### Instalar

Crear un entorno virtual e instalar

```
pip install -r requirements.txt
```

### Ejecutar la app

```
python manage.py migrate
python manage.py runserver
```
